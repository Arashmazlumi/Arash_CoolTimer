# Arash_CoolTimer

A simple, stylish countdown timer built with Python and Tkinter.

## Features

- Set minutes and seconds using spinboxes
- Start, pause, reset, and stop the timer
- Visual countdown with color change (red in last 10 seconds)
- Beep sound when time is up (Windows beep, fallback to system bell)
- Clean and modern dark UI

## Requirements

- Python 3.6 or higher
- Tkinter (usually included with Python)
- No extra libraries required
